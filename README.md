# Моделирование упругого удара шара в вязкой среде
Моделирование физики броска и отскока мяча.<br />
<br />
Использованы технологии графического API WebGL and GLSL.
<br />
<br />
Учитываются следующие физический свойства:
<br />
- сила тяжести;
- сопротивление среды.
